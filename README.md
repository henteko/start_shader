# three.jsではじめるシェーダ入門

技術書典5にて頒布した「three.jsではじめるシェーダ入門」のソースコードです。

## 使い方

このレポジトリをcloneして以下のコマンドを実行してください。

```
$ git clone https://github.com/henteko/start_shader
$ cd start_shader
$ npm install
$ npm start
$ open localhost:3000
```

ブラウザが開いて画面上に男の子の3Dモデルが表示されたら準備完了です。
